# Oauth权限

# 创建访问账号

在M18的[授权应用列表]模块注册一个访问账号(Client ID 和 Client Secret)

# 获取访问权限

使用Client ID、Client Secret 和 能正常登陆M18的用户账号和密码获取访问令牌(access_token)

(请注意：密码是需要经过SHA1加密)
